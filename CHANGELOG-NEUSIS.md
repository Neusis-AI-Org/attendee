# Neusis fork changelog

This file tracks changes that exist in `Neusis-AI-Org/attendee` but **not** in
upstream `attendee-labs/attendee`. Every Neusis-specific commit on `main`
should add a one-line entry here, prefixed with the commit's `[neusis]:`
subject. When upstream-sync PRs conflict with these changes, this list tells
the reviewer what to preserve.

The fork is auto-synced weekly via `.github/workflows/sync-upstream.yml`. Do
not force-push to `main`. Use merge commits, not rebases.

## Format

```
- YYYY-MM-DD <short-sha> [neusis]: <title>
  Why: <one-line motivation>
  Touched: <list of files>
```
